# Women Safety App

## Project Description
This project is a Women Safety Application developed during a hackathon to help women quickly send emergency alerts and stay safe.

## Features
- SOS emergency alert
- User registration
- Emergency contact saving
- Alarm sound for danger situations
- Frontend and backend integration

## Technologies Used
- Python
- Flask
- HTML
- JavaScript
- GitHub

## How it Works
The user registers their details and emergency contact.  
When the SOS button is pressed, the application triggers an alert and sends the user information to the backend system.
